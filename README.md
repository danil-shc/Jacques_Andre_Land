# 🥐 Jacques André — Ремесленная пекарня & Кафе

> Премиальный, отзывчивый и полностью автономный лендинг для крафтовой французской пекарни **Jacques André** (г. Майкоп). Проект разработан с акцентом на журнальную типографику, плавные интерактивные переходы и высокую скорость загрузки.

## ✨ Особенности и функционал

* **Premium Design System:** Строгая палитра из глубоких кофейно-шоколадных оттенков (`#4B2307` для крупных заголовков и `#7E4B30` для акцентов) в сочетании с мягким благородным фоном.
* **Умная навигация и фильтрация:** Интерактивные карточки категорий на главной ("Торты и пирожные", "Завтраки") автоматически перенаправляют в меню с предустановленным активным фильтром.
* **Интеграция Корзины:** Локальная корзина на Vue с возможностью моментального добавления позиций и оформления предзаказа напрямую через WhatsApp/Telegram.
* **100% Self-Contained:** Полное отсутствие внешних сетевых зависимостей (шрифтов, CDN-скриптов и удаленных иконок). Весь UI-пакет работает локально, обеспечивая моментальный рендеринг.

## 🛠 Технологический стек

| Технология | Назначение |
| :--- | :--- |
| **Vue 3 (Composition API)** | Логика интерфейса, реактивность и управление состоянием реактивной корзины |
| **Vite** | Сверхбыстрая сборка проекта и мгновенный HMR (Hot Module Replacement) |
| **Tailwind CSS** | Адаптивная верстка, кастомный оверлей и дизайн-система |
| **Vue Router** | Клиентский роутинг и обработка query-параметров фильтрации |
| **Lucide Vue** | Легкие, премиальные векторные иконки, встроенные локально в бандл |
