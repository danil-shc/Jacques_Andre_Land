# Project Map — Jacques André

Техническая карта проекта для AI-агентов. Документация отражает **фактическое состояние кодовой базы** на момент генерации.

## Назначение

- Быстро ориентироваться в архитектуре без полного сканирования репозитория
- Не ломать зависимости (корзина, роутинг, три паттерна загрузки изображений)
- Сохранять дизайн-систему «Quiet Luxury»

## Оглавление

| Раздел | Путь | Содержание |
|--------|------|------------|
| **Маркетинг** | [marketing/](marketing/README.md) | Концепция, ЦА |
| **Разработка** | [development/](development/README.md) | Стек, экраны, дизайн, фичи, данные, безопасность |

## Ключевые точки входа в код

| Файл | Роль |
|------|------|
| `src/App.vue` | Shell: Header + RouterView + Footer |
| `src/router.js` | Маршруты SPA |
| `src/store/cart.js` | Единственный источник состояния корзины |
| `src/data/products.js` | Каталог товаров (еда + кофе) |
| `src/style.css` | Tailwind v4 `@theme`, шрифты, токены |
| `src/components/ProductCard.vue` | 3D-flip + stepper + варианты кофе |

## Маршруты

```
/           → HomeView (лендинг)
/menu       → MenuView (каталог + фильтры)
/locations  → LocationsView
/cart       → CartView (WhatsApp checkout)
/faq        → FaqView
```

## Критические ограничения для агентов

1. **Нет бэкенда** — заказы уходят через `wa.me`, промокоды захардкожены клиентски
2. **Нет Pinia/Vuex** — только `useCart()` singleton в `src/store/cart.js`
3. **100% self-contained** — без CDN-шрифтов и удалённых ассетов
4. **Три паттерна изображений** — не смешивать без необходимости (см. `development/stack.md`)
